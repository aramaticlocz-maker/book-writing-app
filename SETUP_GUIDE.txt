# 📚 Book Writing App

An AI-powered book writing application that helps authors create cohesive narratives with intelligent character management, story structure preservation, and AI-assisted writing.

## ✨ Features

### 📖 Core Writing Features
- **Smart Chapter Editor**: Real-time editing with word count tracking
- **Grammar & Style Checking**: AI-powered grammar and style validation
- **Story Memory**: Full context maintenance across chapters
- **Version Control**: Track chapter changes and revisions

### 👥 Character Management
- Create detailed character profiles
- Track character consistency across chapters
- Manage character relationships
- Character arc tracking

### 📝 Story Structure
- Outline builder with drag-and-drop interface
- Beat-by-beat organization
- Scene-level planning
- Outline-to-chapter conversion

### 🎨 AI Story Generation
- Genre-specific suggestions
- Chapter continuation generation
- Character dialogue suggestions
- Plot hole detection

### 🎭 Theme Tracking
- Automatic theme identification
- Theme usage tracking across chapters
- Consistency alerts

### 📄 Document Export
- Generate Word documents (.docx)
- PDF export support
- Professional formatting

## 🛠️ Tech Stack

### Frontend
- Next.js 14 + React 18
- TypeScript
- Tailwind CSS

### Backend
- Node.js + Express.js
- PostgreSQL
- OpenAI GPT-4 API

## 🚀 Quick Start

See [docs/SETUP.md](./docs/SETUP.md) for detailed setup instructions.

Happy Writing! 📖✨

import express, { Express } from 'express';
import cors from 'cors';
import dotenv from 'dotenv';
import { initializeDatabase } from './database/connection';
import storyRoutes from './routes/storyRoutes';
import characterRoutes from './routes/characterRoutes';
import chapterRoutes from './routes/chapterRoutes';
import outlineRoutes from './routes/outlineRoutes';
import aiRoutes from './routes/aiRoutes';

dotenv.config();

const app: Express = express();
const PORT = process.env.PORT || 5000;

// Middleware
app.use(cors());
app.use(express.json({ limit: '50mb' }));
app.use(express.urlencoded({ limit: '50mb', extended: true }));

// Initialize database
initializeDatabase();

// Health check
app.get('/health', (req, res) => {
  res.json({ status: '✅ Server is running', timestamp: new Date() });
});

// API Routes
app.use('/api/stories', storyRoutes);
app.use('/api/characters', characterRoutes);
app.use('/api/chapters', chapterRoutes);
app.use('/api/outlines', outlineRoutes);
app.use('/api/ai', aiRoutes);

// Error handling middleware
app.use((err: any, req: any, res: any, next: any) => {
  console.error('Error:', err);
  res.status(500).json({ error: err.message || 'Internal server error' });
});

// Start server
app.listen(PORT, () => {
  console.log(`✅ Server running on port ${PORT}`);
  console.log(`📚 Book Writing App API`);
  console.log(`🔗 http://localhost:${PORT}`);
});

export default app;

import { OpenAI } from 'openai';

export class AIService {
  private openai: OpenAI;

  constructor() {
    this.openai = new OpenAI({
      apiKey: process.env.OPENAI_API_KEY,
    });
  }

  async generateChapterContinuation(
    previousChapters: string[],
    outline: string,
    characters: any[],
    genre: string,
    themes: string[]
  ): Promise<string> {
    const characterContext = characters
      .map(c => `${c.name}: ${c.background}, Personality: ${c.personality}`)
      .join('\n');

    const prompt = `You are a professional ${genre} author skilled in maintaining narrative consistency and character development.

**Previous Chapters Summary:**
${previousChapters.slice(-2).join('\n\n')}

**Main Characters:**
${characterContext}

**Story Themes:** ${themes.join(', ')}

**Outline for This Chapter:**
${outline}

**Task:** Write the next chapter. Ensure:
- Character consistency with previous appearances
- Theme reinforcement
- Proper story pacing
- Vivid, engaging prose
- Proper grammar and spelling

Start writing:`;

    const response = await this.openai.chat.completions.create({
      model: 'gpt-4',
      messages: [{ role: 'user', content: prompt }],
      temperature: 0.7,
      max_tokens: 2000,
    });

    return response.choices[0].message.content || '';
  }

  async checkGrammar(text: string): Promise<any> {
    const prompt = `Proofread this text for grammar, spelling, and style issues. Return a JSON object with:
- "errors": array of {issue: string, suggestion: string}
- "summary": brief summary of main issues

Text: "${text}"`;

    const response = await this.openai.chat.completions.create({
      model: 'gpt-4',
      messages: [{ role: 'user', content: prompt }],
      temperature: 0.3,
    });

    try {
      return JSON.parse(response.choices[0].message.content || '{}');
    } catch {
      return { errors: [], summary: '' };
    }
  }

  async detectThemes(content: string, existingThemes: string[]): Promise<string[]> {
    const prompt = `Identify main themes in this text. Consider these existing themes: ${existingThemes.join(', ')}.
Return as JSON array of theme strings.

Text: "${content.substring(0, 1000)}"`;

    const response = await this.openai.chat.completions.create({
      model: 'gpt-4',
      messages: [{ role: 'user', content: prompt }],
      temperature: 0.5,
    });

    try {
      return JSON.parse(response.choices[0].message.content || '[]');
    } catch {
      return existingThemes;
    }
  }

  async suggestDialogue(
    character: any,
    context: string,
    genre: string
  ): Promise<string> {
    const prompt = `Generate authentic dialogue for this character matching their voice and personality.

Character: ${character.name}
Personality: ${character.personality}
Traits: ${Object.values(character.traits || {}).join(', ')}
Context: ${context}
Genre: ${genre}

Provide 2-3 dialogue options that are natural and character-consistent.`;

    const response = await this.openai.chat.completions.create({
      model: 'gpt-4',
      messages: [{ role: 'user', content: prompt }],
      temperature: 0.8,
    });

    return response.choices[0].message.content || '';
  }

  async detectPlotHoles(
    allChapters: string[],
    characters: any[],
    outline: string
  ): Promise<{ issues: string[]; severity: string[] }> {
    const prompt = `Review this story for plot holes, character inconsistencies, and timeline issues.

Characters:
${characters.map(c => `${c.name}: ${c.background}`).join('\n')}

Story outline: ${outline}

Chapters: ${allChapters.map((ch, i) => `Chapter ${i + 1}: ${ch.substring(0, 500)}...`).join('\n\n')}

Return as JSON with 'issues' array and 'severity' array (low/medium/high).`;

    const response = await this.openai.chat.completions.create({
      model: 'gpt-4',
      messages: [{ role: 'user', content: prompt }],
      temperature: 0.3,
    });

    try {
      return JSON.parse(response.choices[0].message.content || '{"issues":[],"severity":[]}');
    } catch {
      return { issues: [], severity: [] };
    }
  }
}

import Story from '../models/Story';
import Chapter from '../models/Chapter';
import Character from '../models/Character';
import Outline from '../models/Outline';
import Theme from '../models/Theme';

export class StoryService {
  async createStory(data: any) {
    const story = await Story.create(data);
    await Outline.create({ storyId: story.id });
    return story;
  }

  async getStory(id: string) {
    return await Story.findByPk(id);
  }

  async getAllStories() {
    return await Story.findAll();
  }

  async updateStory(id: string, data: any) {
    const story = await Story.findByPk(id);
    if (!story) throw new Error('Story not found');
    return await story.update(data);
  }

  async getStoryWithDetails(id: string) {
    const story = await Story.findByPk(id);
    if (!story) throw new Error('Story not found');

    const chapters = await Chapter.findAll({ where: { storyId: id } });
    const characters = await Character.findAll({ where: { storyId: id } });
    const outline = await Outline.findOne({ where: { storyId: id } });
    const themes = await Theme.findAll({ where: { storyId: id } });

    return {
      ...story.toJSON(),
      chapters,
      characters,
      outline,
      themes,
    };
  }
}

import Chapter from '../models/Chapter';

export class ChapterService {
  async createChapter(data: any) {
    const wordCount = data.content ? data.content.split(/\s+/).length : 0;
    return await Chapter.create({ ...data, wordCount });
  }

  async getChapter(id: string) {
    return await Chapter.findByPk(id);
  }

  async getChaptersByStory(storyId: string) {
    return await Chapter.findAll({ where: { storyId }, order: [['chapterNumber', 'ASC']] });
  }

  async updateChapter(id: string, data: any) {
    const chapter = await Chapter.findByPk(id);
    if (!chapter) throw new Error('Chapter not found');

    if (data.content) {
      data.wordCount = data.content.split(/\s+/).length;
      data.version = (chapter.version || 0) + 1;
    }

    return await chapter.update(data);
  }

  async deleteChapter(id: string) {
    const chapter = await Chapter.findByPk(id);
    if (!chapter) throw new Error('Chapter not found');
    await chapter.destroy();
  }

  async getPreviousChapters(storyId: string, currentChapterNumber: number) {
    return await Chapter.findAll({
      where: { storyId },
      order: [['chapterNumber', 'ASC']],
    });
  }
}

import { Request, Response } from 'express';
import { StoryService } from '../services/StoryService';

const storyService = new StoryService();

export class StoryController {
  async createStory(req: Request, res: Response) {
    try {
      const story = await storyService.createStory(req.body);
      res.status(201).json(story);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async getStories(req: Request, res: Response) {
    try {
      const stories = await storyService.getAllStories();
      res.json(stories);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async getStory(req: Request, res: Response) {
    try {
      const story = await storyService.getStoryWithDetails(req.params.id);
      res.json(story);
    } catch (error: any) {
      res.status(404).json({ error: error.message });
    }
  }

  async updateStory(req: Request, res: Response) {
    try {
      const story = await storyService.updateStory(req.params.id, req.body);
      res.json(story);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }
}

import { Request, Response } from 'express';
import { AIService } from '../services/AIService';
import { ChapterService } from '../services/ChapterService';
import { CharacterService } from '../services/CharacterService';

const aiService = new AIService();
const chapterService = new ChapterService();
const characterService = new CharacterService();

export class AIController {
  async generateChapter(req: Request, res: Response) {
    try {
      const { storyId, genre, themes, outline } = req.body;
      const chapters = await chapterService.getChaptersByStory(storyId);
      const characters = await characterService.getCharactersByStory(storyId);

      const content = await aiService.generateChapterContinuation(
        chapters.map(c => c.summary || c.content),
        outline,
        characters,
        genre,
        themes
      );

      res.json({ content });
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async checkGrammar(req: Request, res: Response) {
    try {
      const { text } = req.body;
      const result = await aiService.checkGrammar(text);
      res.json(result);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async detectThemes(req: Request, res: Response) {
    try {
      const { content, existingThemes } = req.body;
      const themes = await aiService.detectThemes(content, existingThemes);
      res.json({ themes });
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async suggestDialogue(req: Request, res: Response) {
    try {
      const { character, context, genre } = req.body;
      const dialogue = await aiService.suggestDialogue(character, context, genre);
      res.json({ dialogue });
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async detectPlotHoles(req: Request, res: Response) {
    try {
      const { storyId } = req.body;
      const chapters = await chapterService.getChaptersByStory(storyId);
      const characters = await characterService.getCharactersByStory(storyId);
      const result = await aiService.detectPlotHoles(
        chapters.map(c => c.content),
        characters,
        'story outline'
      );
      res.json(result);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }
}

import { Request, Response } from 'express';
import Outline from '../models/Outline';

export class OutlineController {
  async createOutline(req: Request, res: Response) {
    try {
      const outline = await Outline.create(req.body);
      res.status(201).json(outline);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }

  async getOutline(req: Request, res: Response) {
    try {
      const outline = await Outline.findOne({ where: { storyId: req.params.storyId } });
      if (!outline) throw new Error('Outline not found');
      res.json(outline);
    } catch (error: any) {
      res.status(404).json({ error: error.message });
    }
  }

  async updateOutline(req: Request, res: Response) {
    try {
      const outline = await Outline.findOne({ where: { storyId: req.params.storyId } });
      if (!outline) throw new Error('Outline not found');
      await outline.update(req.body);
      res.json(outline);
    } catch (error: any) {
      res.status(500).json({ error: error.message });
    }
  }
}

import axios from 'axios';
import { Story, Chapter, Character, Outline } from '@/types/story';

const API_URL = process.env.NEXT_PUBLIC_API_URL || 'http://localhost:5000/api';

const api = axios.create({
  baseURL: API_URL,
  headers: {
    'Content-Type': 'application/json',
  },
});

export const storyAPI = {
  createStory: async (story: Partial<Story>) => api.post<Story>('/stories', story),
  getStories: async () => api.get<Story[]>('/stories'),
  getStory: async (id: string) => api.get<Story & { chapters: Chapter[]; characters: Character[] }>(`/stories/${id}`),
  updateStory: async (id: string, story: Partial<Story>) => api.put<Story>(`/stories/${id}`, story),
};

export const chapterAPI = {
  createChapter: async (chapter: Partial<Chapter>) => api.post<Chapter>('/chapters', chapter),
  getChapter: async (id: string) => api.get<Chapter>(`/chapters/${id}`),
  getChaptersByStory: async (storyId: string) => api.get<Chapter[]>(`/chapters/story/${storyId}`),
  updateChapter: async (id: string, chapter: Partial<Chapter>) => api.put<Chapter>(`/chapters/${id}`, chapter),
  deleteChapter: async (id: string) => api.delete(`/chapters/${id}`),
};

export const characterAPI = {
  createCharacter: async (character: Partial<Character>) => api.post<Character>('/characters', character),
  getCharacter: async (id: string) => api.get<Character>(`/characters/${id}`),
  getCharactersByStory: async (storyId: string) => api.get<Character[]>(`/characters/story/${storyId}`),
  updateCharacter: async (id: string, character: Partial<Character>) => api.put<Character>(`/characters/${id}`, character),
  deleteCharacter: async (id: string) => api.delete(`/characters/${id}`),
  checkConsistency: async (id: string) => api.get(`/characters/${id}/consistency`),
};

export const outlineAPI = {
  getOutline: async (storyId: string) => api.get<Outline>(`/outlines/${storyId}`),
  updateOutline: async (storyId: string, outline: Partial<Outline>) => api.put<Outline>(`/outlines/${storyId}`, outline),
};

export const aiAPI = {
  generateChapter: async (data: any) => api.post('/ai/generate-chapter', data),
  checkGrammar: async (text: string) => api.post('/ai/check-grammar', { text }),
  detectThemes: async (content: string, existingThemes: string[]) => api.post('/ai/detect-themes', { content, existingThemes }),
  suggestDialogue: async (character: any, context: string, genre: string) => api.post('/ai/suggest-dialogue', { character, context, genre }),
  detectPlotHoles: async (storyId: string) => api.post('/ai/detect-plot-holes', { storyId }),
};

export default api;

import Link from 'next/link';

  const [loading, setLoading] = useState(true);
  const [showNewStory, setShowNewStory] = useState(false);
  const [newStory, setNewStory] = useState({ title: '', genre: '', description: '', authorId: 'user-1' });

  useEffect(() => {
    fetchStories();
  }, []);

  const fetchStories = async () => {
    try {
      const response = await storyAPI.getStories();
      setStories(response.data);
    } catch (error) {
      console.error('Failed to fetch stories:', error);
    } finally {
      setLoading(false);
    }

  const handleCreateStory = async (e: React.FormEvent) => {
    e.preventDefault();
    try {
      await storyAPI.createStory(newStory);
      setNewStory({ title: '', genre: '', description: '', authorId: 'user-1' });
      setShowNewStory(false);
      fetchStories();
    } catch (error) {
      console.error('Failed to create story:', error);
    }
  };

  if (loading) {
    return (
      <div className="min-h-screen bg-gradient-to-br from-blue-50 to-indigo-100 flex items-center justify-center">
        <div className="text-center">
          <div className="text-4xl mb-4">📚</div>
          <p className="text-xl text-gray-700">Loading your stories...</p>
        </div>
      </div>
    );
  }

  return (

      {/* Header */}
      <header className="bg-white shadow">
        <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 py-6">
          <div className="flex justify-between items-center">

            >
              + New Story
            </button>
          </div>
        </div>

        <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 py-6">
          <div className="bg-white rounded-lg shadow-lg p-6">

# Setup Guide

## Prerequisites

- Node.js 18+
- PostgreSQL 14+
- OpenAI API key

## Installation

### 1. Backend Setup

```bash
cd backend
npm install
cp .env.example .env
```

Update `.env` with your database credentials and OpenAI API key.

### 2. Database Setup

```bash
# Using Docker (recommended)
docker run --name book-db -e POSTGRES_PASSWORD=password -d -p 5432:5432 postgres:15

# Create database
createdb book_writing_app
```

### 3. Start Backend Server

```bash
cd backend
npm run dev
```

Server will run on `http://localhost:5000`

### 4. Frontend Setup

```bash
cd frontend
npm install
cp .env.example .env
```

### 5. Start Frontend

```bash
cd frontend
npm run dev
```

App will be available at `http://localhost:3000`

## Environment Variables

### Backend (.env)

```
PORT=5000
NODE_ENV=development
DATABASE_URL=postgresql://user:password@localhost:5432/book_writing_app
OPENAI_API_KEY=sk-...
JWT_SECRET=your-secret-key
```

### Frontend (.env.local)

```
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```


# Architecture Overview

## Project Structure

```
book-writing-app/
├── backend/           # Node.js/Express API
├── frontend/          # Next.js React app
└── docs/              # Documentation
```

## Backend Architecture

### Models
- **Story**: Main story container with title, genre, description
- **Character**: Character profiles with personality, relationships, arc tracking
- **Chapter**: Story chapters with content, themes, character tracking
- **Outline**: Story structure and planning with scenes
- **Theme**: Story themes tracking usage across chapters

### Services
- **StoryService**: Story CRUD and full details retrieval
- **CharacterService**: Character management and consistency checking
- **ChapterService**: Chapter management with version control
- **AIService**: OpenAI integration for generation and analysis

### Controllers
- REST endpoints for all resources
- Error handling and validation
- Response formatting

## Frontend Architecture

### Pages
- `/`: Stories list and creation
- `/story/[id]`: Story detail view
- `/story/[id]/editor`: Chapter editor (future)
- `/story/[id]/characters`: Character management (future)
- `/story/[id]/outline`: Outline builder (future)

### Services
- `api.ts`: Axios client for backend communication
- API methods for all resources

### Types
- TypeScript interfaces for all data models

## Data Flow

```
Frontend UI → API Service → Express Routes → Controllers → Services → Database
           ← Response ←     ← Models ←    ← Business Logic ←
```

## AI Integration

- OpenAI GPT-4 for story generation
- Grammar and style checking
- Theme detection and analysis
- Character dialogue suggestions
- Plot hole detection

# API Documentation

## Base URL

```
http://localhost:5000/api
```

## Stories Endpoints

### Create Story
```
POST /stories

Body:
{
  "title": "My Story",
  "genre": "Fantasy",
  "description": "Story description",
  "authorId": "user-1"
}
```

### Get All Stories
```
GET /stories
```

### Get Story Detail
```
GET /stories/:id
```

Returns story with all chapters, characters, outline, and themes.

### Update Story
```
PUT /stories/:id

Body:
{
  "title": "Updated Title",
  "description": "Updated description"
}
```

```
Body:
```

```
Body:
```

```
  "outline": "Chapter outline..."
}

}
```

### Check Grammar
```
POST /ai/check-grammar

Body:
{
  "text": "Text to check..."
}

Response:
{
  "errors": [{ "issue": "...", "suggestion": "..." }],
  "summary": "Summary"
}
```

### Detect Themes
```
POST /ai/detect-themes

Body:
{
  "content": "Chapter content...",
  "existingThemes": ["theme1", "theme2"]
}

  "themes": ["theme1", "theme2", "newTheme"]
}
```

```
}
```
